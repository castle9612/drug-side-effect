# Drug Side Effect Research Assets

본 저장소는 전체 연구 개요 문서에 사용되는 결과 요약표와 그래프 자산을 모아둔 폴더이다. 발표자료 기준 최종 비교 결과와 보존된 CSV 기반 세부 실험 요약을 함께 포함한다.

## English Summary

This repository stores the summary tables and figures used in the top-level research overview README.

## 포함 자료

| 폴더 | 설명 |
| --- | --- |
| `metrics` | 실험 요약 CSV |
| `figures` | 비교 그래프 및 ablation figure |

## 주요 파일

| 파일 | 설명 |
| --- | --- |
| `metrics/presentation_final_model_summary.csv` | 2024-02-06 발표자료 기준 최종 모델 비교 |
| `metrics/major_experiment_summary.csv` | 보존된 주요 실험 로그 요약 |
| `figures/presentation-final-comparison.svg` | 발표자료 기준 최종 비교 그래프 |
| `figures/xgboost-ablation.svg` | XGBoost 세부 비교 그래프 |
