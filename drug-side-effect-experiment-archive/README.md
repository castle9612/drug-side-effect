# Drug Side Effect Experiment Archive

이 저장소는 연구 초기에 시도한 여러 프로토타입, 중간 산출물, 발표 시기 작업 파일을 보관한 아카이브이다. 정제된 메인 레포와 달리, 연구 과정의 시행착오와 아이디어 변화를 보여주는 작업 공간에 가깝다.

## English Summary

This repository is an archive of exploratory prototypes, intermediate outputs, and working materials from the early stages of the study. It preserves the research process rather than presenting a polished pipeline.

## 연구 개요

| 항목 | 내용 |
| --- | --- |
| 연구 초점 | 초기 시도, 초안 코드, 작업 기록 보관 |
| 입력 데이터 | `simliarity_action.csv`, `result.csv`, 발표 자료 등 |
| 수행 내용 | 다양한 초기 모델 및 특징 생성 아이디어 탐색 |
| 저장소 역할 | 연구 과정 아카이브 |

## 주요 파일

| 파일 | 설명 |
| --- | --- |
| `model.py` | 초기 모델 프로토타입 |
| `rf_0417.py` | 초기 RF 관련 코드 |
| `xgboost_ae.py` | AE 기반 XGBoost 시도 |
